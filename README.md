# ROTUM Protocol 🚀

> Trust, contribution, and portable on-chain reputation infrastructure for The Open Network (TON).

ROTUM Protocol turns verified ecosystem participation into measurable digital credentials. It provides anti-Sybil frameworks, dynamic contribution engine telemetry, and contributor profile frameworks designed to reward genuine long-term community members.
